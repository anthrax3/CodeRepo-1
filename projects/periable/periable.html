Periable

A Periodic Table App in JavaScript.

Description

Periable is a periodic table app written in JavaScript. It not only works as a normal periodic table but also has some extra functionalities.

It also contains a Compound Identifier, Molar Mass Calculator & Stochiometric Percentage Calculator. That's not it. It also has one more functionality, The Scale. It's primary function is to order the elements in periodic table according to the features given below.

Scales

  • Melting Point
  • Boiling Point
  • Electronegativity
  • Mass
  • Density
  • Specific Heat

And more....

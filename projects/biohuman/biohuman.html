Biohuman

A Human Character Animator for JavaScript.

Description

Biohuman is a human character animator in JavaScript. It supports both male & female characters.

Whatever you'll want them to do, they'll do. But you'll have to code the respective functionalities. Some of them are prebuilt. It also supports Green-Screen video recording.

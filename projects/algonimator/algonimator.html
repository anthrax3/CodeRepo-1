Algonimator

An Algorithm Animator for JavaScript.

Description

Algonimator is an algorithm animator written in JavaScript. Its primary aim is to show how a specific algorithm works. To achieve that, it simply traces the program and shows it in form of an array/graph/chart.

It is very simple to use & supports every type of algorithms. Currently, there are very few but most significant algorithms. User can also write his own algorithm inTry your Algorithmsection.
